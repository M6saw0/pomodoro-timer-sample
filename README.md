# ポモドーロタイマー PWA

オフライン対応のプログレッシブ・ウェブ・アプリケーション（PWA）版ポモドーロタイマーです。

## 概要

ポモドーロ・テクニックをサポートする、モバイルファースト設計のタイマーアプリです。複数のタイマーを連続実行できるシーケンス機能で、作業と休憩のサイクルを柔軟に管理できます。

### 主な機能

- **タイマーシーケンス設定** - 複数のタイマーを連続実行可能
- **プリセットテンプレート** - クラシック(25分/5分)、ショート(15分/3分)、ロング(50分/10分)
- **カスタマイズ機能** - タイマー時間、ラベル、サイクル数を自由に設定
- **リピート制御** - 1〜10回のサイクル繰り返し
- **開始・一時停止・再開・リセット** - 直感的なタイマーコントロール
- **音声アラート + バイブレーション** - タイマー終了時に通知
- **PWA対応** - オフライン動作、ホーム画面へのショートカット追加可能
- **Wake Lock機能** - タイマー実行中は画面スリープを防止
- **レスポンシブUI** - モバイル・タブレット・デスクトップに対応
- **ダークモード** - OSの設定に自動追従
- **自動保存** - タイマー設定はlocalStorageに自動保存

## デプロイ

### Vercelへのデプロイ（無料・推奨）

#### 1. Vercelアカウント作成（初回のみ）

[Vercel.com](https://vercel.com) にアクセスして、無料アカウントを作成します。GitHubアカウントでログインするのが簡単です。

#### 2. GitHubリポジトリにpush

```bash
git add .
git commit -m "Initial commit"
git push origin main
```

#### 3. Vercelでプロジェクトをインポート

Vercel ダッシュボードで「Add New... > Project」をクリック。

GitHubリポジトリを選択し、`Import` をクリック。

#### 4. デプロイ設定（デフォルトで自動設定されます）

- **Framework**: Vite
- **Build Command**: `npm run build`
- **Output Directory**: `dist`

「Deploy」をクリックすると、自動的にビルド・デプロイが開始されます。

#### 5. デプロイ完了

デプロイが完了すると、Vercelから一意のURLが発行されます。

以降、`main` ブランチへのpushは自動的にデプロイされます。

## iPhone/Safari での使用方法

### ホーム画面に追加（PWA化）

iPhoneのSafariからアプリをホーム画面に追加すると、ネイティブアプリのように使用できます。

#### 手順

1. **Safari でアプリを開く**

   デプロイされたURLをブラウザで開きます。

2. **共有ボタンをタップ**

   画面下部の共有ボタン（□と↑）をタップします。

3. **「ホーム画面に追加」をタップ**

   メニューが表示される場合は、「ホーム画面に追加」を探してタップします。

4. **タイトルを確認して「追加」をタップ**

   デフォルトで「ポモドーロタイマー」と表示されます。必要に応じて名前を変更できます。

5. **完了**

   ホーム画面にアプリアイコンが追加され、タップするとSafariブラウザなしで起動できます。

#### PWAのメリット

- **オフライン対応** - インターネット接続がなくても、一度読み込んだページは使用可能
- **高速起動** - ネイティブアプリと同じ速度で起動
- **画面スリープ防止** - Wake Lock API対応で、タイマー実行中は画面が消えない
- **通知機能** - 音声とバイブレーションで終了を通知

## 使い方

### 基本的な操作

1. **アプリを起動**

   ホーム画面のアイコンから起動、またはブラウザでアクセス。

2. **タイマーを選択**

   プリセット（クラシック・ショート・ロング）をタップするか、カスタムで時間を設定。

3. **リピート回数を設定（オプション）**

   1〜10回の中から選択。デフォルトは4回。

4. **「スタート」をタップ**

   タイマーが開始されます。

5. **操作**

   - **一時停止** - 実行中に「一時停止」ボタンをタップ
   - **再開** - 「再開」ボタンをタップ
   - **リセット** - 「リセット」ボタンをタップして初期状態に戻す

6. **終了通知**

   タイマーが終了すると、音声とバイブレーションで通知。次のステップが自動開始されます。

### プリセット説明

- **クラシック 25/5** - 作業25分 + 休憩5分（標準的なポモドーロ・テクニック）
- **ショート 15/3** - 作業15分 + 休憩3分（短時間の集中向け）
- **ロング 50/10** - 作業50分 + 休憩10分（深い集中作業向け）

各プリセットは4回繰り返されます。

### カスタム設定

**タイマー構成** セクションで自由に設定可能：

- **ラベル** - 各ステップに名前を付ける（例：「集中」「ストレッチ」）
- **時間** - 1〜99分の範囲で設定
- **リピート** - サイクルの繰り返し回数（1〜10回）

設定は自動的に保存されます。

## トラブルシューティング

### アプリが立ち上がらない

ブラウザキャッシュをクリアしてリロードしてください。

```
Safari: 設定 > プライバシー > ウェブサイトデータ > [このサイト]を削除
Chrome: 設定 > キャッシュされた画像とファイル > 削除
```

### タイマーが止まってしまう

- **画面ロック設定** - Wake Lock API がサポートされていない環境では、画面のスリープを防止できません。スリープ設定を延長してください。
- **バッテリーセーバー** - バッテリーセーバーがオンの場合、アプリが一時停止することがあります。オフにしてください。

### ホーム画面に追加できない

iOS 12.2以降で対応しています。Safari以外のブラウザ（Chrome、Firefoxなど）では、「ホーム画面に追加」がサポートされていません。Safari をご使用ください。

### 音が出ない

- **音量設定** - デバイスの音量がミュートになっていないか確認
- **マナーモード** - マナーモードがオンになっていないか確認
- **ブラウザ設定** - ブラウザがマイクロフォン・オーディオの使用を許可しているか確認
